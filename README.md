# 15_ServiceLayer_Practice_Product

A simple and clean Product Management System built using ASP.NET Core MVC, Entity Framework Core, SQL Server, and Service Layer Architecture.

## Features

* Create Product
* View Product List
* Update Product Information
* Delete Product
* Search Products by Name
* Form Validation
* Clean Service Layer Implementation
* Responsive Bootstrap UI

## Technologies Used

* ASP.NET Core MVC
* C#
* Entity Framework Core
* SQL Server
* Bootstrap 5
* LINQ

## Project Structure

```text
Controllers
│
├── ProductsController

Models
│
├── Product

Services
│
├── ProductService

Data
│
├── ApplicationDbContext

Views
│
├── Product
```

## Service Layer Implementation

Business logic is separated from the controller using a dedicated Service Layer.

Benefits:

* Better Code Organization
* Maintainability
* Reusability
* Easy Testing
* Industry Standard Architecture

## Search Functionality

Users can search products directly from the product listing page using product name.

Implemented using LINQ filtering.

## CRUD Operations

### Create

Add new products with validation.

### Read

Display all products in a structured table.

### Update

Edit existing product details.

### Delete

Safely remove products from the system.
